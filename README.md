## 🤖 AI Agents 一览

- **[Fellou（Agentic 浏览器）](https://fellou.ai/)**  
  全球首个 Agentic 浏览器，支持智能浏览、自动执行任务、跨应用集成，适合高效信息处理和自动化办公。
- **[Manus（通用 AI Agent）](https://manus.im/usecase-official-collection)**
